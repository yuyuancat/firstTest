# 简答题
## 1. 简述浏览器请求数据时，同步和异步的区别？
- 同步处理：代码执行到同步处理的代码时会执行该代码，不继续执行后面的代码
- 异步处理：代码执行到对应异步执行的代码块/代码时，会同时继续执行后面的代码

## 2. 简述 ajax 的过程。
1. 创建XMLHttpRequest对象
2. 设置需要访问的url和访问方式(open())
3. 执行访问（send()）
4. 执行状态变化事件处理函数

## 3. 请解释一下 JavaScript 的同源策略。
- 同域名
- 同端口
- 同协议

## 4. GET和POST的区别，何时使用POST？
- GET：获取数据时使用，请求参数会拼接到url后面，明文传输
- POST：添加数据时使用，请求参数放在请求体里面，需要设置请求头中的"content-Type"

## 5. 什么是jsonp跨域。
通过script标签发送跨域请求，服务端返回一个带参数的全局函数调用

## 6. 什么是cors跨域。
这种方案无须客户端做出任何变化，只是在被请求的服务端响应的时候添加一个Access-Control-Allow-Origin的响应头，表示这个资源是否允许指定域请求。

## 7. 请描述jsonp 与 cors 的区别。
- cors跨域不需要客户端修改逻辑代码
- jsonp的全局函数需要客户端去定义。服务端给这个函数进行传参

## 8. 简单描述模板的作用。
- 整洁代码，使代码不凌乱
- 减少性能开销

## 9. 请描述ArtTemplate模板的使用步骤。
1. 下载ArtTemplate代码
2. 将脚本引入HTML文件
3. 创建模板标签（script），标签需要有type 和 id
4. 使用template(id, args),进行使用模板
